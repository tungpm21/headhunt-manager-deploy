## 🟡 Warnings (Nên sửa)

### W1. npm vulnerability: `defu` (High severity)

- **Vấn đề:** Package `defu` có lỗ hổng Prototype Pollution (GHSA-737v-mqg7-c878)
- **Nguy hiểm:** Hacker có thể inject property vào object JavaScript, gây lỗi logic. Tuy nhiên `defu` là dependency gián tiếp (transitive), không được gọi trực tiếp trong code — rủi ro thực tế **THẤP**.
- **Cách sửa:** `npm audit fix`

### W2. Pre-existing TypeScript errors trong `candidate-actions.ts`

- **File:** `src/lib/candidate-actions.ts` (line 22, 26, 29)
- **Vấn đề:** S5 agent thay đổi Prisma types/job query với `mode: "insensitive"` nhưng dùng `string` thay vì cast `as const`, gây mismatch với Prisma types
- **Nguy hiểm:** Build PASS (Next.js compiles OK), nhưng strict `tsc --noEmit` sẽ fail. Không ảnh hưởng runtime.
- **Cách sửa:**
```typescript
// Line 26-29: Thêm "as const" cho mode
{ title: { contains: query, mode: "insensitive" as const } }
```

### W3. Sprint 3 log entries thiếu dấu tiếng Việt

- **File:** `docs/PROJECT-TRACKER.md` (lines 252-286)
- **Vấn đề:** Các log entries do S5 agent ghi đều không có dấu tiếng Việt

- **Gợi ý:** Bỏ `previewFeatures = ["driverAdapters"]` trong `schema.prisma` nếu không dùng adapter mode
