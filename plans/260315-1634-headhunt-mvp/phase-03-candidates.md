# Phase 03: Candidate Management
Status: ⬜ Pending
Dependencies: Phase 02

## Mục tiêu
Xây dựng toàn bộ tính năng quản lý ứng viên - phần CORE của app.

## Tasks

### CRUD Ứng viên
1. [ ] Trang danh sách ứng viên (table view, phân trang)
2. [ ] Trang thêm ứng viên mới (form đầy đủ)
3. [ ] Trang chi tiết ứng viên (xem + sửa)
4. [ ] Xóa ứng viên (soft delete - ẩn đi, không xóa hẳn)

### Upload CV
5. [ ] Upload file CV (PDF/Word, max 10MB)
6. [ ] Hiển thị link download CV trong hồ sơ
7. [ ] Thay thế CV (upload lại)

### Tìm kiếm & Lọc
8. [ ] Tìm kiếm theo tên, SĐT, email
9. [ ] Lọc theo ngành, vị trí, khu vực
10. [ ] Lọc theo mức lương (khoảng min-max)
11. [ ] Lọc theo trạng thái (Available, Đã có việc...)
12. [ ] Lọc theo tags

### Tags & Ghi chú
13. [ ] Gắn tags cho ứng viên (chọn có sẵn hoặc tạo mới)
14. [ ] Thêm ghi chú (text + timestamp + ai viết)
15. [ ] Xem lịch sử ghi chú

### Thông tin ứng viên cần lưu
- Họ tên, SĐT, email, ngày sinh, giới tính
- Vị trí hiện tại, công ty hiện tại
- Ngành nghề, số năm kinh nghiệm
- Mức lương hiện tại, mức lương mong muốn
- Khu vực (TP.HCM, Hà Nội...)
- Trạng thái: Available / Đã có việc / Đang interview / Blacklist
- Nguồn: LinkedIn, TopCV, Giới thiệu, Khác
- CV file
- Tags
- Ghi chú (nhiều ghi chú, có timestamp)

## Output
- Quản lý ứng viên đầy đủ, dễ dùng
- Upload & lưu CV
- Tìm kiếm + lọc nhanh
- Ghi chú & tags
